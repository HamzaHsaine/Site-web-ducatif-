HTML & CSS Débutant

Bienvenue dans ce cours d’introduction à HTML et CSS. Vous allez apprendre à créer une page web simple et à la styliser.

1. Qu’est-ce que HTML ?

HTML (HyperText Markup Language) est le langage qui structure une page web. Il organise le contenu en titres, paragraphes, listes, images, liens, etc.

2. Structure de base d’une page HTML

Voici un exemple minimaliste d’une page HTML :

<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="UTF-8">
    <title>Ma page web</title>
  </head>
  <body>
    <h1>Bienvenue sur mon site</h1>
    <p>Ceci est un paragraphe.</p>
  </body>
</html>

3. Les balises principales

  • <h1> ... </h1> : Titre principal
  • <p> ... </p> : Paragraphe
  • <a href="url"> ... </a> : Lien
  • <img src="image.jpg" alt="description"> : Image
  • <ul> ... </ul> : Liste non ordonnée
  • <li> ... </li> : Élément de liste

4. Ajouter du style avec CSS

CSS (Cascading Style Sheets) permet de personnaliser l’apparence des éléments HTML.

Exemple : changer la couleur du texte et la police du titre :

h1 {
  color: #f1c40f;
  font-family: 'Arial Black', sans-serif;
}

5. Exemple complet : page HTML simple avec style CSS

<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="UTF-8">
    <title>Page Stylée</title>
    <style>
      body {
        background: #f0f8ff;
        font-family: Arial, sans-serif;
        color: #333;
        padding: 20px;
      }
      h1 {
        color: #3498db;
      }
      p {
        font-size: 1.1rem;
      }
    </style>
  </head>
  <body>
    <h1>Bonjour le monde !</h1>
    <p>Ceci est une page web simple avec du style CSS.</p>
  </body>
</html>

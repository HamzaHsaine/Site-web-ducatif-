Cours JavaScript pour Tous

Bienvenue dans ce cours d'introduction à JavaScript. Nous allons découvrir les bases qui vous permettront de rendre vos pages web interactives.

1. Syntaxe et Variables

JavaScript utilise des variables pour stocker des données. Voici comment déclarer des variables :

// Déclaration de variables
let nom = "Alice";
const age = 25;
var ville = "Paris";
Exemple : Afficher un message dans la console
console.log("Bonjour, " + nom + " !");

2. Fonctions et Événements

Les fonctions permettent de regrouper des instructions réutilisables :

function saluer(nom) {
  alert("Bonjour, " + nom + " !");
}

Vous pouvez appeler cette fonction ainsi :

saluer("Alice");
Exemple : Bouton avec un événement click

3. Manipulation du DOM

Le DOM (Document Object Model) permet d’interagir avec la page HTML :

// Modifier le contenu d’un élément
document.getElementById('monTexte').textContent = "Texte mis à jour !";

Ce texte va changer.

4. Introduction à AJAX

AJAX permet de charger des données sans recharger la page :

fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => {
    console.log(data);
  });

Ceci est un aperçu rapide, le sujet sera développé plus tard.

Bonne continuation dans votre apprentissage de JavaScript !

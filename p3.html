Cours Python pour Débutants

Bienvenue dans ce cours d'introduction à Python, un langage simple et puissant pour débuter la programmation.

1. Variables et Types

En Python, on peut stocker des données dans des variables :

nom = "Alice"
age = 25
taille = 1.75  # en mètres
est_etudiant = True
Exemple : Afficher des informations avec print()
print("Nom :", nom)
print("Âge :", age)

2. Structures conditionnelles

On utilise if pour prendre des décisions :

if age >= 18:
    print("Adulte")
else:
    print("Mineur")

3. Boucles

Pour répéter des actions, on utilise for ou while :

for i in range(5):
    print("Itération", i)

4. Fonctions

On définit des fonctions pour regrouper du code :

def saluer(nom):
    print("Bonjour", nom, "!")

saluer("Alice")
Exemple : Fonction simple
def carre(nombre):
    return nombre * nombre

print(carre(4))  # Affiche 16

Continuez à pratiquer et explorez les nombreuses possibilités de Python !
